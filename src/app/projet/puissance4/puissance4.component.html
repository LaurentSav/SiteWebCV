Description du projet


Projet scolaire portant sur la conception du jeu 'Puissance 4' et l'implémentation d'un agent basé sur l'utilité.
Cet agent utilise l'algorithme Min-Max avec élégage Alpha-Beta pour choisir le meilleur coup selon la pronfondeur de l'arbre de recherche donnée.
